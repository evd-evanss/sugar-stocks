# Sugar Stocks
Um aplicativo para acompanhar e monitorar dados de ações na B3 

## API utilizada
https://brapi.dev/

## Organização de módulos
    app (presentation)
        |
        -- data
        -- domain
        -- design-system

    data
        |
        -- domain

    domain

    design-system

## Principais Frameworks
- Jetpack Compose (UI)
- Testes (Mockk)
- Injeção de dependências (Hilt)
- Dados Reativos (Kotlin Flow)
